## Getting Started

第一步：在根目录创建 `.env.local` 文件，填入：

```
LEANCLOUD_APPID=Leancloud的AppId
LEANCLOUD_APPKEY=Leancloud的AppKey
```

第二步：

[数据清洗相关的工作](./scripts/elleria/README.md)，必须等待数据完成同步

第三步：

run the development server

```bash
npm run dev
# or
yarn dev
```
